在Win7下安装步骤

下载MongoDB

Download
下载Windows 32-bit版本并解压缩，程序文件都在bin目录中，其它两个目录分别是C++调用是的头文件和库文件。bin目录中包含如下几个程序：
 mongo.exe，命令行客户端工具。  
 mongod.exe，数据库服务程序。  
 mongodump.exe，数据库备份程序。  
 mongoexport.exe，数据导出工具。  
 mongofiles.exe，GridFS工具。  
 mongoimport.exe，数据导入工具。  
 mongorestore.exe，数据库恢复工具。  
* mongos.exe，貌似是性能检测工具。

MongoDB目录

将其解压到 d:\，再重命名为mongodb，路径为d:\mongodb

设置数据文件路径

在d:\mongodb盘建一个data文件夹，在data文件夹中新建db文件夹，路径d:\mongodb\data\db;
在mongodb文件夹下建立log文件夹，并在log文件夹下建立mongodb.log文件

启动MongoDB服务

进入 cmd 提示符控制台：

 日志文件以及添加方式记录（追加）
 数据目录为d:\MongoDB\data\db，并且每个数据库将储存在一个单独的目录（–directoryperdb）

MongoDB服务端的默认连接端口:27017
http://localhost:27017/
可以看到如下提示：
 You are trying to access MongoDB on the native driver port. For http diagnostic access, add 1000 to the port number
如此，MongoDB数据库服务已经成功启动了

客户端连接验证

新打开一个CMD输入：d:\mongodb\bin\mongo，如果出现下面提示，那么您就可以开始MongoDB之旅了：

查看MongoDB日志

查看D:\mongodb\log\mongodb.log文件，即可对MongoDB的运行情况进行查看或排错。
