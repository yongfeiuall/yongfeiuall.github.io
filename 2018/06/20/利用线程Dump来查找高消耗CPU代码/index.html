利用线程Dump来查找高消耗CPU代码

Posted on 2018-06-20 | In Performance Testing | |

对于CPU消耗高的代码，可以用下面的方法来进行定位分析。

前提

运行一段Java代码，让它消耗CPU。

定位

  1. 用top查找出高消耗CPU PID

  2. 用top -p <PID>查找出线程ID - (输入H查看所有线程的统计信息)

  3. 用jstack <PID>做线程Dump

  4. 转换（十进制 - 十六进制）nid

  5. 在Dump中查找相应nid

基本上一定位一个准。
